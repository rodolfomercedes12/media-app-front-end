# media_content_app


## Configuraración inicial

1. Ejecutar en la terminal el comando "flutter pub get".
2. Iniciar un emulador
3. Ejecutar en la terminal el comando "flutter run" o presionar la tecla  f5 para lanzar la aplicación.
